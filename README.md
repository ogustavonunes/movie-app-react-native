# 🎬 Movie App - React Native

Aplicação desenvolvida em React Native + Expo para demonstrar:

- Navegação entre telas
- React Navigation
- Consumo de API REST
- Listagem de filmes

## 📱 Tecnologias utilizadas

- React Native
- Expo
- React Navigation
- JavaScript
- Fetch API

---

## 🚀 Funcionalidades

✅ Home Screen  
✅ Details Screen  
✅ Navegação Stack  
✅ Consumo de API REST  
✅ Lista de filmes  
✅ Interface estilizada

---

## 📂 Estrutura do projeto

```bash
src/
 └── screens/
      ├── HomeScreen.js
      ├── DetailsScreen.js
      └── MoviesScreen.js
```

---

## ▶️ Como executar

### Instalar dependências

```bash
npm install
```

### Rodar projeto

```bash
npx expo start
```
